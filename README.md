# CNN

## 一、实验目的
- 复用 CNN 模型完成 MNIST 分类，比较不同优化器（SGD、SGD+Momentum、Adam）及学习率（0.1、0.01、0.001）的效果。
- 可视化第一层卷积核和特征图，分析错误样本，绘制混淆矩阵。

## 二、实验环境
Python 3.7+，PyTorch、torchvision、matplotlib、numpy、scikit-learn。

## 三、数据集
MNIST 手写数字（28×28 灰度），训练集 48000 张，验证集 12000 张，测试集 10000 张。归一化处理（均值 0.1307，标准差 0.3081）。

## 四、CNN 模型结构
- Conv1(1→32, 3×3) + ReLU + MaxPool2d(2)  
- Conv2(32→64, 3×3) + ReLU + MaxPool2d(2)  
- Flatten → FC(3136→128) → ReLU → FC(128→10)

## 五、实验结果

### 5.1 优化器对比（5 epoch）
| 优化器          | 测试准确率 |
|----------------|-----------|
| SGD            | 97.83%    |
| SGD+Momentum   | 98.84%    |
| Adam           | 98.86%    |

结论：Adam 与带动量 SGD 收敛快、精度高，优于普通 SGD。

### 5.2 学习率对比（Adam）
| 学习率 | 测试准确率 | 现象         |
|--------|-----------|--------------|
| 0.1    | 9.80%     | 不收敛       |
| 0.01   | 96.52%    | 收敛慢、波动 |
| 0.001  | 99.04%    | 最佳         |

结论：学习率过大导致梯度爆炸，过小收敛慢；推荐 0.001。

### 5.3 错误样本分析
测试集准确率 99.03%，错误 97 张。最易混淆数字对：6→5（11次）、7→9（7次）、3→5（6次）、9→5（6次）、2→7（5次）。原因：低分辨率下手写风格相似。改进：数据增强、加深网络、正则化。

### 5.4 卷积核与特征图
- 第一层卷积核呈现边缘、方向、斑点纹理，通过反向传播自动学习。
- 不同特征图对图像的水平、垂直、对角笔画等区域响应不同，提取了多样化低层特征。

### 5.5 混淆矩阵
- 对角线：正确分类样本数（越高越好）。
- 非对角线：错误样本数，最大值为(6,5)，6 和 5 最易混淆。
